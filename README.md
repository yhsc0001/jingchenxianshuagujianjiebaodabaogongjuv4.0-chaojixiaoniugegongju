## 简介
本仓库提供了一个用于晶晨系盒子线刷固件解包和打包的工具，版本为 v4.0，由超级小牛哥开发。该工具可以帮助用户轻松处理卡刷包和线刷包，适用于需要对固件进行解包或打包操作的用户。

## 使用说明
1. **卡刷包**：
   - 文件格式为 `.zip` 后缀。
   - 将 `.zip` 文件直接放在工具的根目录下即可。

2. **线刷包**：
   - 文件格式为 `.img` 后缀。
   - 将 `.img` 文件放在工具目录内的 `in` 文件夹中。
